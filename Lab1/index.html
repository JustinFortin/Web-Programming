Basic concepts of web technology

1. Why do we need a Uniform Resource Locator (URL)?

We need URLs because it is a way for websites to store its location as well as retrieve it

2. Give an example of Uniform Resource Locator (URL) and specify all components of an URL.

Example: https://www.bestbuy.ca/en-ca

Scheme
https:// describes the protocol involved when retrieving the contents of the page or file.
Host
www.bestbuy.ca is the name of the host, Best Buy, from the World Wide Web (www).
Path
/en-ca is the path to the Canadian english version of the website.
Query String
Not in this URL but ?fid=2830034 is an example of a possible query string.

3. What is the difference between a static and a dynamic web site?

The difference between a static web site and a dynamic one is that a static website only contains
HTML pages that are identical on every platform while a dynamic web site has its contents created at
run-time by a program written by a programmer.

4. Why do we need Domain Name System (DNS)?

We need a Domain Name System (DNS) because without it surfing the web would become impractical
and very frustrating. DNS is the reason people can type in www.google.ca instead of having to remember
the specific IP address of every website you want to visit.

5. What is DNS address resolution?

DNS resolution is the process of transforming the domain name to the corresponding IP address.

8. What is the function (purpose) of Hypertext Transfer Protocol (HTTP)?

The purpose of HTTP protocol is to securely transfer data to the user on the World Wide Web.

9. Give some examples of HTTP request methods.

  • GET
  • PUT
  • POST
  • PATCH

10. Define a Web Server.

A Web Server is a server that runs software to process clients requests for websites using HTTP protocol as well
as some other similar ones depending on the site.
