What are the main benefits of using CSS?

Some of the main benefits of using CSS is to make the webpage(s) easier to update and maintain, allows the markup language file(s) to be a lot lighter in code and less cluttered, and includes many more options than just simple markup.

What are different parts of a CSS style rule

A CSS style rule has 2 parts: a selector and a declaration.

What are an element selector and grouped element selector in CSS? Provide an example of each.

An element selector in CSS is to select the content you want to select to edit such as : h1 { color: red; }
A grouped element in CSS is to select multiple elements that will have the same attributes to minimize code such as: h1, h2, p { text-align: center; color: red;}

What are class selectors in CSS? What are id selectors in CSS?

Class selectors in CSS are used to select elements of a class attribute and begin with a period followed by the class name.
ID selectors in CSS are used to select a single instance of an element. This allows more flexibility to control each tag individually. They begin with a #.

What are contextual selectors in CSS?

Contextual selectors in CSS are used to specify styles for several parts of the file.

What are the three cascade principles used by browsers when style rules conflict?

The three main principles used by browsers is Inheritance, the Cascade, and Specificity.
